# Proyecto para prueba de diagnostico

## Descripción Técnica

El proyecto fue realizado con PHP 8.5.1, la ultima version, y con Postgresql 18, la ultima igualmente.

## Instalación

Para hacerlo funcionar, crear archivo .env en la carpeta raiz, con los siguientes valores:

```DB_HOST=
DB_PORT=
DB_NAME=
DB_USER=
DB_PASS=
```

Luego es ejecutar:

` php -S localhost:8000`
